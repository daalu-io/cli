Daalu CLI

The daalu coding agent for your terminal.

Linux / macOS

curl -fsSL https://get.daalu.io/install.sh | sh

Windows (PowerShell)

irm https://get.daalu.io/install.ps1 | iex

Homebrew

brew install daalu-io/tap/daalu

Then:

daalu login
cd ~/your/project && daalu

Binaries: github.com/daalu-io/cli/releases
